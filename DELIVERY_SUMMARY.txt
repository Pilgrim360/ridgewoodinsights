IMPLEMENTATION OVERVIEW
================================================================================

GOAL:
Reposition the editor toolbar from a sticky position inside the TipTapEditor
component to a fixed position immediately beneath the CMS header, ensuring
the toolbar remains visible while users scroll through content.

SOLUTION:
Uses React Context (CmsHeaderSlotsContext) to inject the toolbar into the
layout hierarchy, positioning it above the scrollable content area with
proper z-index stacking.

ARCHITECTURE:
- EditorToolbarGlobal: New component that manages toolbar injection
- CmsHeaderSlotsContext: Extended with toolbar slot
- CmsLayout: Renders toolbar slot from context
- TipTapEditor: Uses EditorToolbarGlobal instead of internal toolbar
- CSS: Enhanced globals.css with toolbar styling

KEY BENEFITS:
✓ Toolbar always accessible while editing
✓ Modern, intuitive user experience
✓ Seamless integration with existing code
✓ Full responsive support (desktop, tablet, mobile)
✓ Zero breaking changes
✓ Proper memory management
✓ Well-documented and maintainable
